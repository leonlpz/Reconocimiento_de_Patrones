# Codigos en Python con ejemplos y ejercicios del curso 'Reconocimiento de Patrones'

Introducción a los conceptos básicos del análisis y reconocimiento de patrones. 
El curso aborda esquemas lineales y no lineales del aprendizaje supervisado y no supervisado, orientando el uso de herramientas computacionales para el procesamiento, análisis y toma de decisiones a partir de datos. 
Además, se presenta una introducción a los conceptos fundamentales del aprendizaje profundo.
